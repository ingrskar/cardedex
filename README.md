# Cardédex

A Pokémon card collection app built with React and TypeScript.

## Features

- Search for Pokémon with autocomplete
- View Pokémon cards with stats and details
- Catch Pokémon and add them to your collection (with confetti!)
- Browse your personal collection

## Tech Stack

- [React 19](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/)
- [Vite](https://vitejs.dev/) — build tool
- [React Router 7](https://reactrouter.com/) — client-side routing
- [Emotion](https://emotion.sh/) — CSS-in-JS styling
- [SWR](https://swr.vercel.app/) — data fetching
- [PokéAPI](https://pokeapi.co/) — Pokémon data

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Scripts

| Command         | Description              |
| --------------- | ------------------------ |
| `npm run dev`   | Start development server |
| `npm run build` | Build for production     |
| `npm run preview` | Preview production build |
| `npm run lint`  | Run ESLint               |
